# NLP Basics Repository

- `datasets/` → Kullanılan veri setlerini içerir.
- `notebooks/` → Çeşitli NLP tekniklerini içeren Jupyter Notebook'lar.
  - `classification/` → Metin sınıflandırma modelleri.
